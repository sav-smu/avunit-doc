**********
Types
**********

AVUnit is a dynamic typed language, which means we can directly assign values to variables.
AVunit provides several elementary types which is useful for scenario description and specification defination. 

In addition, types can interact with each other in expressions containing
operators. For a quick reference of the various operators, see .
